XSLT Test

Download the XML file I have linked here from our class's Frank O'Hara Project: McDermottOHaraPoemRevision.xml. The file contains a poem with some interesting markup that we'd like you to transform into HTML, according to our instructions:

Overview: This test will require you to write modal XSLT to produce, first, an HTML reading view of this complete poem according to our instructions below, followed by an unordered list of the pop culture references marked inside it (coded as <ref what="popCulture">.

The HTML presentation of the poem should meet the following specifications:

  1. Your HTML must be output as a valid XHTML strict file, which means you must have the appropriate xsl:output statement to set this doctype definition.
  2. Each stanza of the poem (marked as <st>) and the date following the poem, should be held inside an HTML <p> element.
  3. Each line of the poem should be output with an HTML break element, <br/>, and also so that its line number is visible and coded to be styled in CSS differently from the main text, like this: <span class="number">2</span>
  4. The poem features some interesting examples of words broken across lines, so they are literally broken in half by a line break. Your HTML presentation should transform the contents of the wordBreak elements so they appear bold in the HTML, using the HTML <b> element.
  5. Let's make the contents of the refrain elements stand out by coding them with italics, using the HTML <i> element.
  6. Write a template rule to transform all the ref elements, using <span class=""> (which would allow us to style this a particular way in CSS), and use an attribute value template to determine the value of @class. (You don't need to write CSS for this test.)
  7. Now, write modal XSLT to output an unordered (bulleted) list to appear after the reading view of the poem, using <ul> and <li> to list the contents of the Pop Culture references in this poem.

Check your work! Be sure to open your HTML file in <oXygen/> to be sure it is valid and well-formed, and modify your XSLT accordingly. When you are finished, upload only your XSLT file in Courseweb in "Tests" at the designated upload point for the XSLT exam. We only need to see your XSLT file.
